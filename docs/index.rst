=====================================
ScreenPy Appium: A ScreenPy Extension
=====================================

Not sure what ScreenPy is?
Check out `ScreenPy's documentation <https://screenpy-docs.readthedocs.io/en/latest/>`__ first!

``screenpy_appium`` extends ScreenPy
by providing Abilities,
Actions,
Questions,
and Resolutions
which interact with `Appium <https://appium.io//>`__.
This enables your Actors
to :class:`~screenpy_appium.abilities.UseAnAndroidDevice`
and :class:`~screenpy_appium.abilities.UseAnIOSDevice`!
